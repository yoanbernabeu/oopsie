# oopsie-sdk

User-initiated bug reporting SDK for [Oopsie](https://github.com/yoanbernabeu/oopsie).

## Install

```bash
npm install oopsie-sdk
```

Or via CDN:

```html
<script src="https://unpkg.com/oopsie-sdk"></script>
```

## Usage

```js
import Oopsie from 'oopsie-sdk';

Oopsie.init({
  serverUrl: 'https://oopsie.yourapp.com',
  apiKey: 'your-project-key',
});
```

## Documentation

Full documentation available at: https://yoanbernabeu.github.io/oopsie/docs/sdk/installation
